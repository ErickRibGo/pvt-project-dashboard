# PVT Project Dashboard

Projeto técnico para o processo seletivo de Desenvolvedor Web e Infraestrutura JR da PVT Software & Serviços.

## Objetivo

Construir um módulo de painel de projetos capaz de acompanhar horas vendidas, planejadas e realizadas, saldo de horas, avanço físico, avanço financeiro e saúde dos projetos.

## Stack inicial

- Frontend: React + TypeScript
- Backend: Node.js + TypeScript + Express
- Banco de dados: PostgreSQL
- ORM: Prisma
- Testes: Vitest
- Infraestrutura local: Docker Compose
